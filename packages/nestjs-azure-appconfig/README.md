## Description

`@digikare/nestjs-azure-appconfig` is a nestjs module that integrate [Azure App Configuration](https://docs.microsoft.com/fr-fr/azure/azure-app-configuration).

## Installation

```bash
npm i --save @digikare/nestjs-azure-appconfig
```
